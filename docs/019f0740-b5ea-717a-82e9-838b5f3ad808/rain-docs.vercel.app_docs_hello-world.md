Rain programs live in `.rn` files. Statements end with a semicolon. Strings use double quotes.

Build the interpreter, then run your file from the terminal — same idea as `node script.js`.

hello.rn

```
print "Hello, Rain!";
```

output

```
Hello, Rain!
```

## Run it

Run it

```
make
./rain hello.rn
```

output

```
Hello, Rain!
```
