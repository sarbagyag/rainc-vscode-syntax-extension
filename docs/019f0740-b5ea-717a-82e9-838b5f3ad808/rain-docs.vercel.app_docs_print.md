`print` evaluates an expression and writes it to stdout, followed by a newline.

You can print strings, numbers, booleans, `nil`, arrays, and other values.

## Basic output

Basic output

```
print "Hello";
print "World";
print 42;
print #[1, 2];
```

output

```
Hello
World
42
[1, 2]
```
