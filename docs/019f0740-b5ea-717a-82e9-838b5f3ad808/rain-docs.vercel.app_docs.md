## Get started

- [Hello World](https://rain-docs.vercel.app/docs/hello-world) Your first Rain program — save a file and run it.
- [Build & CLI](https://rain-docs.vercel.app/docs/install) Compile Rain from source and run scripts.

## Basics

- [Variables](https://rain-docs.vercel.app/docs/variables) Declare mutable bindings with \`var\`.
- [Types & literals](https://rain-docs.vercel.app/docs/types) Dynamic typing — numbers, strings, booleans, arrays, and \`nil\`.
- [Operators](https://rain-docs.vercel.app/docs/operators) Arithmetic, comparison, logic, and assignment.
- [Print](https://rain-docs.vercel.app/docs/print) Show output with \`print\`.
- [Comments](https://rain-docs.vercel.app/docs/comments) Line comments with \`//\`.
- [Arrays](https://rain-docs.vercel.app/docs/arrays) Literal arrays, indexing, and methods.

## Control flow

- [If / else](https://rain-docs.vercel.app/docs/if-else) Branch on conditions — parentheses required.
- [While loops](https://rain-docs.vercel.app/docs/while) Repeat while a condition is true.
- [For loops](https://rain-docs.vercel.app/docs/for) C-style \`for (init; condition; increment)\` loops.

## Functions

- [Functions](https://rain-docs.vercel.app/docs/functions) Define reusable code with \`fun\`.
- [Closures](https://rain-docs.vercel.app/docs/closures) Functions capture variables from surrounding scopes.

## Classes

- [Classes](https://rain-docs.vercel.app/docs/classes) Objects with methods, fields, and constructors.
- [Inheritance](https://rain-docs.vercel.app/docs/inheritance) Extend classes with \`<\` and call superclass methods with \`super\`.

## Modules

- [Importing](https://rain-docs.vercel.app/docs/importing) Load modules with \`benutzen\` and call \`name::function()\`.
- [math](https://rain-docs.vercel.app/docs/math) Trigonometry, rounding, random, and array math.
- [str](https://rain-docs.vercel.app/docs/str) String search, slice, split, join, and format.
- [json](https://rain-docs.vercel.app/docs/json) Parse and stringify JSON.
- [http](https://rain-docs.vercel.app/docs/http) HTTP server, responses, client, and CORS.
- [db](https://rain-docs.vercel.app/docs/db) SQLite — open, exec, query with parameters.
- [io](https://rain-docs.vercel.app/docs/io) Read, write, append files; read stdin.
- [os](https://rain-docs.vercel.app/docs/os) Platform info, environment, sleep, exit.
- [time](https://rain-docs.vercel.app/docs/time) Current date, time parts, formatting.

## Reference
